# InkOS Web Frontend

This folder is reserved for the assistant-ui based web console.

Planned scope:
- Chief Editor workspace at `/chief`
- Book, chapter, truth-file, automation, and settings pages
- Global floating assistant entry using `AssistantModal`
- Unified assistant-ui runtime backed by a local InkOS service layer

Implementation details are documented in:
- `docs/frontend-requirements.md`
- `docs/frontend-design.md`
